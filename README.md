# .dotfiles
A repository to track changes in my dotfiles.

# Getting started

Clone the repo to your home directory. ($HOME).

```bash
  git clone https://github.com/chuxmykel/.dotfiles.git
```

# Installation
First you need to make the installation script executable.

```bash
chmod +x ./install.sh
```

Then run the script

```bash
./install.sh
```
